# CodeSensei Documentation

Welcome to the official documentation for the **CodeSensei Platform**.

CodeSensei is an all-in-one platform for Computer Science mastery. It provides tools to:
- Generate structured practical reports from your code
- Convert between various document formats (PDF, DOCX, Markdown, LaTeX)
- Practice for vivas and interviews with AI-generated questions
- Visualize Data Structures and Algorithms
- Analyze codebases and generate architecture diagrams

## Getting Started

To get started with development, please review the [Setup Guide](setup.md) and [Architecture](architecture.md) documentation.

For detailed information regarding individual subsystems, check the side navigation menu.
